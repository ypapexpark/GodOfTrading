# LinguaFlow Android 친구 테스트 안내

아래 APK 파일 하나를 친구에게 전송한다.

`LinguaFlowMobile/android/app/build/outputs/apk/debug/app-debug.apk`

카카오톡에서 APK 전송이 차단되면 Google Drive, Dropbox, 이메일 또는 USB 파일 전송을 사용한다.

## 친구에게 그대로 보내는 안내 문구

> 내가 개발 중인 LinguaFlow 테스트를 부탁할게. 영어 문장을 번역하거나 교정하려고 다른 앱에 복사·붙여넣기하지 않고, 현재 입력창에서 바로 처리하는 키보드 앱이야.
>
> 1. 전달한 `app-debug.apk`를 휴대폰에 저장하고 파일을 눌러줘.
> 2. “알 수 없는 앱 설치” 안내가 나오면, APK를 연 앱(Chrome·내 파일 등)에 대해서만 잠시 설치를 허용해줘.
> 3. Play Protect 경고가 나오면 앱 이름이 `LinguaFlow`인지 확인하고 테스트 설치를 진행해줘. 회사 관리 휴대폰처럼 설치가 막힌 기기에서는 중단해도 돼.
> 4. LinguaFlow를 실행하고 기본 번역 언어를 `한국어`로 선택해줘.
> 5. 먼저 앱 아래쪽의 `바로 시험해 보기`에서 영어 문장을 입력하고 `Google 번역`을 눌러봐.
> 6. 그다음 `키보드 활성화`를 눌러 설정에서 `LinguaFlow 키보드`를 켜줘. Android가 “키보드가 입력 내용을 수집할 수 있다”는 공통 경고를 보여주는 것은 정상 동작이야. 이 테스트 버전은 유료 서버에 문장을 보내지 않고 기기 내 모델로 처리하고, 비밀번호 입력란에서는 기능을 끄도록 만들었어.
> 7. 메모·메신저 입력창에서 지구본 모양 키보드 전환 버튼을 눌러 `LinguaFlow 키보드`를 선택해줘.
> 8. `I have a meeting tomorrow.` 같은 문장을 입력하고 키보드 위의 `Google 번역`을 눌러봐. 같은 입력창에서 한국어로 바뀌면 성공이야.
> 9. 기존 삼성/Gboard 키보드를 계속 쓰고 싶으면 문장을 길게 눌러 선택한 뒤 `더보기` → `LinguaFlow로 다듬기`를 찾아봐. 이 메뉴는 일부 앱에서는 표시되지 않을 수 있어.
>
> 테스트가 끝나면 지구본 버튼으로 원래 키보드로 돌아오면 돼. 원하면 Android 설정 → 키보드 목록에서 LinguaFlow를 다시 끌 수도 있어.

## 꼭 받아야 할 피드백

친구에게 아래 항목을 답해 달라고 한다.

- 휴대폰 제조사와 모델명
- Android 버전
- 테스트한 앱 이름(삼성 메모, 카카오톡, WhatsApp 등)
- `Google 번역` 버튼을 눌렀을 때 같은 입력칸에서 교체됐는지
- 문장 선택 메뉴에 `LinguaFlow로 다듬기`가 나타났는지
- 교정·다듬기를 눌렀을 때 성공했는지, 미지원 안내가 나왔는지
- 불편했던 단계와 화면 캡처 또는 화면 녹화

## 테스트 시 알아둘 제한

- 설치 가능한 최소 버전은 Android 8.0이다.
- 현재 키보드는 영문 QWERTY V1이다. 한글을 직접 조합해 입력하는 기능은 아직 없다.
- 번역 모델은 최초 실행 시 인터넷으로 다운로드되므로 첫 번역이 오래 걸릴 수 있다.
- 교정·다듬기는 Gemini Nano를 지원하는 일부 기기에서만 동작한다.
- 은행·보안 앱은 제3자 키보드를 차단할 수 있다.
- 이 파일은 앱스토어 배포본이 아니라 개발용 디버그 APK다.

## 테스트 종료 후 삭제

Android 설정 → 앱 → LinguaFlow → 삭제를 누른다. 설치할 때 허용했던 “알 수 없는 앱 설치” 권한도 Chrome 또는 파일 앱 설정에서 다시 끌 수 있다.
